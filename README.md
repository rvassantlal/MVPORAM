# myPathORAM

# How to run
**Note:** Use ``./smartrun.sh`` instead of ``.\smartrun.cmd`` in Unix-based operating system.
#### With BFT-SMaRt
1. Start servers.
    ```
    .\smartrun.cmd pathoram.Server <process id>
    ```
2. Start client.
    ```
    .\smartrun.cmd pathoram.ClientInterface <client id>
    ```
